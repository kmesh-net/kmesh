# 集群节点说明

## 功能概述

集群节点页展示当前 Kubernetes 集群中所有节点的基本信息与状态。

## 展示字段

- **节点名称**：节点在集群中的名称
- **状态**：节点就绪状态（Ready / NotReady）
- **角色**：节点角色（如 control-plane、worker）
- **内网 IP**：节点内网 IP 地址
- **运行时长**：节点运行时间
- **内核**：节点内核版本
- **OS 镜像**：节点操作系统镜像

## 使用说明

1. 点击顶部导航「集群节点」进入
2. 点击「刷新」按钮可重新拉取节点列表
3. 数据来源于当前 kubeconfig 所指向的集群
