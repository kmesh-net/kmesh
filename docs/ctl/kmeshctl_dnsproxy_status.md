## kmeshctl dnsproxy status

Display the current DNS proxy status

```bash
kmeshctl dnsproxy status [podNames...] [flags]
```

### Examples

```bash
kmeshctl dnsproxy status
kmeshctl dnsproxy status pod1 pod2
```

### Options

```bash
  -h, --help   help for status
```
