## kmeshctl dnsproxy enable

Enable DNS proxy for Kmesh

```bash
kmeshctl dnsproxy enable [podNames...] [flags]
```

### Examples

```bash
kmeshctl dnsproxy enable
kmeshctl dnsproxy enable pod1 pod2
```

### Options

```bash
  -h, --help   help for enable
```
