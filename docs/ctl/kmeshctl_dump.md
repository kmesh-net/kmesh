## kmeshctl dump

Dump config of kernel-native or dual-engine mode

```bash
kmeshctl dump [flags]
```

### Examples

```bash
# Kernel Native mode:
kmeshctl dump <kmesh-daemon-pod> kernel-native

# Dual Engine mode:
kmeshctl dump <kmesh-daemon-pod> dual-engine
```

### Options

```bash
  -h, --help   help for dump
```
