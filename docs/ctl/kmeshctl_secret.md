## kmeshctl secret

Use secrets to generate secret configuration data for IPsec

```
kmeshctl secret [flags]
```

### Examples

```
# Use secrets to generate secret configuration data for IPsec:
 kmeshctl secret aeadAlgo aeadKey aeadLength
```

### Options

```
  -h, --help   help for secret
```
