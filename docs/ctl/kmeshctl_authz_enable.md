## kmeshctl authz enable

Enable xdp authz eBPF program for Kmesh's authz offloading

```
kmeshctl authz enable [podNames...] [flags]
```

### Examples

```
kmeshctl authz enable
kmeshctl authz enable pod1 pod2
```

### Options

```
  -h, --help   help for enable
```
