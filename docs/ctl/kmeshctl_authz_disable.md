## kmeshctl authz disable

Disable xdp authz eBPF program for Kmesh's authz offloading

```
kmeshctl authz disable [podNames...] [flags]
```

### Examples

```
kmeshctl authz disable
kmeshctl authz disable pod1 pod2
```

### Options

```
  -h, --help   help for disable
```
