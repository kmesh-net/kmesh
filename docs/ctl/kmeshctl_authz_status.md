## kmeshctl authz status

Display the current authorization status

```bash
kmeshctl authz status [podNames...] [flags]
```

### Examples

```bash
kmeshctl authz status
kmeshctl authz status pod1 pod2
```

### Options

```bash
  -h, --help   help for status
```
