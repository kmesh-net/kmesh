## kmeshctl dnsproxy disable

Disable DNS proxy for Kmesh

```bash
kmeshctl dnsproxy disable [podNames...] [flags]
```

### Examples

```bash
kmeshctl dnsproxy disable
kmeshctl dnsproxy disable pod1 pod2
```

### Options

```bash
  -h, --help   help for disable
```
