sidebar_position: 3

Dump config of ads-v1 or ads-v2 mode

```bash
kmeshctl dump [flags]
```

### Examples

```bash
# Ads-v1 mode:
kmeshctl dump <kmesh-daemon-pod> ads-v1

# Ads-v2 mode:
kmeshctl dump <kmesh-daemon-pod> ads-v2
```

### Options

```bash
  -h, --help   help for dump
```
